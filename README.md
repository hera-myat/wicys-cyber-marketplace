# WICYS CYBER MARKETPLACE

A single-scroll, interactive React website built for **WiCyS UIC** during a winter break
pair-programming series.

This project focuses on a creative, user-friendly interface, smooth scrolling interactions, and
portfolio-quality frontend development using modern web tools.

## üõ† TECH STACK

- **React.js**: A component-based JavaScript library for building dynamic and reusable user interfaces, chosen for its flexibility and industry relevance.

- **Vite**: A fast, modern build tool and development server that offers instant startup and hot module replacement, allowing us to iterate quickly and focus on creativity rather than configuration.

- **HTML/CSS/JavaScript**: Core web technologies used to structure, style, and add interactivity to the site.

- **Figma**: A collaborative design tool used to create and share the UI/UX design before implementation.  

- **GitHub**: Version control and collaboration platform used for managing branches, pull requests, and team workflows.
