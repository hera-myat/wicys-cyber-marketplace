A single-scroll, interactive React website built for **WiCyS UIC** during a winter break
pair-programming series.

This project focuses on creative UI design, smooth scrolling interactions, and
portfolio-quality frontend development using modern web tools.

### Daily Workflow
Before starting work each time:
```bash

```

After making changes:
```bash

git push
```
This pushes only to your branch, not main.
